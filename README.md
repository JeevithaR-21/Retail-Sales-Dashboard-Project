Retail Sales Dashboard Project

Project Overview

This project analyzes retail sales data using Microsoft Excel. The objective is to transform raw sales data into meaningful business insights through Pivot Tables, charts, KPIs, and an interactive dashboard.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Lookup Functions
- Dashboard Design

Dataset

The dataset contains retail sales information, including:

- Products
- Categories
- Salespersons
- States
- Sales
- Profit
- Quantity

Dashboard Features

- Product Quantity Analysis
- Sales by Salesperson
- Profit by State
- Sales by Category
- KPI Cards (Total Sales, Total Profit, Total Quantity)
- Business Insights Dashboard

Skills Demonstrated

- Data Cleaning
- Data Analysis
- Pivot Tables
- Data Visualization
- Dashboard Creation
- Business Insight Generation

Project Outcome

This dashboard provides a clear overview of retail sales performance, helping identify top-performing products, salespersons, categories, and profitable states for better business decision-making.
